    Нужен LibreOffice

    CueDeck использует LibreOffice для конвертации PPTX, PPT и KEY файлов в PDF.

    Вариант 1 — скачать установщик
    Вариант 2 — через Homebrew (терминал)
    brew install --cask libreoffice

    После установки перезапусти CueDeck.
